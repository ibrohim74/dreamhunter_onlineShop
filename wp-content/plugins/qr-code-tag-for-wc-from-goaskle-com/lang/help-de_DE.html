1. Größe und Ränder

Der Bereich des QR-Code-Symbols erfordert einen Rand oder eine "Ruhezone" um ihn herum, damit er verwendet werden kann. Der Rand ist ein freier Bereich um ein Symbol, in dem nichts gedruckt wird. Der QR-Code erfordert mindestens einen vier QR-Code-Pixel breiten Rand an allen Seiten eines Symbols.

Wenn Sie die Bildgröße Ihres QR-Codes manuell einstellen, kann es vorkommen (je nach Kapazität des Inhalts), dass das kodierte Bild "pixelig" ist (die Größe eines QR-Code-Pixels variiert) und somit für Ihr Lesegerät schwer zu lesen ist. Dies geschieht in der Regel, wenn Ihr QR-Code-Bild sehr klein ist (kleiner als 100 Pixel). Um ein Bild im "Best Read Mode" auszudrucken, verwenden Sie das Größenfeld, um eine "Zoom"-Stufe (0-9, mit 0=kein Zoom, 1=100%, 2=200%, usw.) für eine optimale QR-Code-Bildgröße Ihres Inhalts anzugeben.

2. Verwendung von Shortcodes

Im Beitragseditor gibst du ein

[qrcodetag_from_goaskle_com/] ein und ein QR-Code mit einem Link zur aktuellen URL wird gerendert
[qrcodetag_from_goaskle_com]content[/qrcodetag_from_goaskle_com], um einen QR-Code mit Inhalt

Sie können jeden Standardwert überschreiben, indem Sie ihn als Parameter im Shortcode übergeben, z.B.
[qrcodetag_from_goaskle_com size="150" link="true"]content[/qrcodetag_from_goaskle_com] rendert einen QR-Code mit einer Größe von 150px und setzt einen Link auf das QR-Code-Bild.

In order to use the tooltip function, type this code

[qrcodetag_from_goaskle_com tooltip="QR Code content"]your link or text[/qrcodetag_from_goaskle_com]

whereas QR Code content is the QR Code image content and your link or text is the visible text - mind the difference to the previous usage!

3. PHP Function usage

To use the QR Code Tag inside your template or PHP enabled posts, use the global $qrcodetag_from_goaskle_com object:

global $qrcodetag_from_goaskle_com;
echo $qrcodetag_from_goaskle_com->getQrCodeUrl($content, $size, $encoding, $ecc, $margin, $version);

Um einen 100 x 100px großen QR-Code in der Seitenvorlage anzuzeigen, würde man folgenden PHP-Code einfügen:

<img src="<?php 
  global $qrcodetag_from_goaskle_com; 
  echo $qrcodetag_from_goaskle_com->getQrCodeUrl('mycontent',100,'UTF-8','L',4,0); 
?>">

4. Verwendung des Widgets

Um den QR-Code-Tag als Widget zu nutzen, fügen Sie ihn einfach in Ihre Sidebar ein (Aussehen | Widgets) und stellen die Optionen entsprechend ein.

5. Bekannte Probleme

- Problem mit selbstschließenden Tags (siehe Shortcode-Warnung)
- Problem mit gemischten Tags (siehe Shortcode-Warnung)
- Problem mit leeren Tags (siehe Shortcode-Warnung)
- Falsche Randgröße (nur mit Google API)
