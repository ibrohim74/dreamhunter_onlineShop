1. Taille et marge

La zone du symbole du code QR nécessite une marge ou une " zone de silence " autour d'elle pour être utilisée. La marge est une zone claire autour d'un symbole où rien n'est imprimé. Le code QR nécessite une marge d'au moins quatre pixels de largeur de code QR de tous les côtés d'un symbole.

Si vous définissez manuellement la taille de l'image de votre QR Code, il peut arriver (en fonction de la capacité du contenu) que l'image encodée soit "floue en pixels" (la taille d'un pixel de QR Code varie) et donc difficile à lire pour votre appareil de lecture. Cela se produit généralement si la taille de votre image de code QR est très petite (inférieure à 100 pixels). Afin d'imprimer une image en "mode de lecture optimale", utilisez le champ de taille pour spécifier un niveau de "Zoom" (0-9, avec 0=pas de zoom, 1=100%, 2=200%, etc. ) pour une taille d'image de code QR optimale de votre contenu.

2. utilisation du shortcode

Dans l'éditeur de posts, tapez

[qrcodetag_from_goaskle_com/] et un code QR avec un lien vers l'URL actuel sera rendu
. [qrcodetag_from_goaskle_com]content[/qrcodetag_from_goaskle_com] pour rendre un code QR avec contenu
.

Vous pouvez écraser n'importe quelle valeur par défaut en la passant comme paramètre dans le shortcode, par exemple
. [qrcodetag_from_goaskle_com size="150" link="true"]content[/qrcodetag_from_goaskle_com] rendra un QR code avec une dimension de 150px et définit un lien vers l'image du QR code.

Pour utiliser la fonction tooltip, tapez ce code

[qrcodetag_from_goaskle_com tooltip="Contenu du code QR"]votre lien ou texte[/qrcodetag_from_goaskle_com]

où QR Code content est le contenu de l'image du QR Code et votre lien ou texte est le texte visible - attention à la différence avec l'usage précédent !

3. utilisation des fonctions PHP

Pour utiliser la balise QR Code à l'intérieur de votre modèle ou des posts compatibles avec PHP, utilisez l'objet global $qrcodetag_from_goaskle_com:

global $qrcodetag_from_goaskle_com;

echo $qrcodetag_from_goaskle_com->getQrCodeUrl($content, $size, $encoding, $ecc, $margin, $version);
.

Afin d'afficher un code QR de 100 x 100px dans le modèle de page, vous devriez inclure ce code PHP :

. 
<img src="<?php 
  global $qrcodetag_from_goaskle_com ; 
  echo $qrcodetag_from_goaskle_com->getQrCodeUrl('mycontent',100, 'UTF-8', 'L',4,0) ; 
?>">

4. utilisation du widget

Afin d'utiliser le QR Code Tag comme un widget, il suffit de le déposer sur votre barre latérale (Apparence | Widgets) et de définir les options en conséquence.
