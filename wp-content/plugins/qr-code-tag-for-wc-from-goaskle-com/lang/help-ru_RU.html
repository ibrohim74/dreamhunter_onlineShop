1. Размер и границы

Для использования области символа QR-кода необходимо наличие поля или "тихой зоны" вокруг него. Поле - это чистая область вокруг символа, на которой ничего не печатается. QR Code требует, чтобы со всех сторон символа было по меньшей мере четыре пикселя шириной.

Если вы вручную установите размер изображения вашего QR-кода, может случиться так (в зависимости от объема содержимого), что закодированное изображение будет "размыто на пиксели" (размер одного пикселя QR-кода варьируется) и, таким образом, будет трудночитаемым для вашего устройства чтения. Обычно это происходит, если размер изображения QR-кода очень мал (менее 100 пикселей). Чтобы распечатать изображение в режиме наилучшего считывания, используйте поле "Размер" для указания уровня "Масштаба" (0-9, где 0=без масштаба, 1=100%, 2=200% и т.д.) для оптимального размера изображения QR-кода вашего содержания.

2. Использование шорткодов

В редакторе постов введите

[qrcodetag_from_goaskle_com/] и QR-код со ссылкой на текущий URL будет выдан
[qrcodetag_from_goaskle_com]содержание[/qrcodetag_from_goaskle_com] для отображения QR-кода с содержанием

Вы можете перезаписать любое значение по умолчанию, передав его в качестве параметра в шорткоде, например.
[qrcodetag_from_goaskle_com size="150" link="true"]содержание[/qrcodetag_from_goaskle_com] отобразит QR-код с размером 150px и устанавливает ссылку на изображение QR-кода.

Чтобы использовать функцию всплывающей подсказки, введите следующий код

[qrcodetag_from_goaskle_com tooltip="Содержание QR-кода"]ваша ссылка или текст[/qrcodetag_from_goaskle_com]

в то время как Содержание QR-кода это содержание изображения QR-кода и ваша ссылка или текст видимый текст - обратите внимание на разницу с предыдущим использованием!

3. Использование функций PHP

Чтобы использовать метку QR-кода внутри вашего шаблона или Посты с поддержкой PHP, использовать глобальный $qrcodetag_from_goaskle_com объект:

global $qrcodetag_from_goaskle_com;
echo $qrcodetag_from_goaskle_com->getQrCodeUrl($content, $size, $encoding, $ecc, $margin, $version);

Для того чтобы отобразить 100 x 100px QR-код в шаблоне страницы вы должны включить этот PHP-код:

<img src="<?php 
  global $qrcodetag_from_goaskle_com; 
  echo $qrcodetag_from_goaskle_com->getQrCodeUrl('mycontent',100,'UTF-8','L',4,0); 
?>">

4. Использование виджетов

Чтобы использовать QR Code Tag в качестве виджета, просто поместите его на боковую панель (Appearance | Widgets) и задайте соответствующие параметры.

5. Известные проблемы

- проблема с самозакрывающимся тегом (см. предупреждение шорткода)
- проблема смешанных тегов (см. предупреждение о шорткодах)
- проблема пустого тега (см. предупреждение о шорткоде)
- неправильный размер полей (только для Google API)
